# Тестирование Trello REST API

Пет-проект по ручному тестированию REST API сервиса Trello.

## Цель проекта

Практика тестирования API, составления тестовой документации и работы с Postman.

## Объект тестирования

- Boards API
- Lists API
- Cards API
- Actions API

## Инструменты

- Postman
- Trello REST API
- GitHub
- JSON
- HTTP
