# Тестирование Trello REST API

Пет-проект по ручному тестированию REST API сервиса Trello.

## Цель проекта

Практика тестирования REST API, разработки тестовой документации, проверки функциональности API и анализа соответствия фактического поведения ожидаемому.

## Объект тестирования

В рамках проекта выполняется тестирование следующих ресурсов Trello REST API:

- Boards API
- Lists API
- Cards API
- Actions API

## Проверяемые сценарии

В проекте реализованы:

- позитивные тестовые сценарии;
- негативное тестирование;
- проверка обязательных параметров;
- проверка граничных значений;
- проверка обработки ошибок API;
- проверка CRUD-операций.

## Инструменты

- Postman — отправка запросов и проверка ответов API;
- Trello REST API — объект тестирования;
- GitHub — хранение проекта и тестовой документации;
- JSON — анализ структуры ответов;
- HTTP — проверка методов и статус-кодов.
