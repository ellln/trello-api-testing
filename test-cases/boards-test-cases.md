# Тест-кейсы: Boards API

## TC-BOARD-001 — Получение списка досок авторизованного пользователя

**Приоритет:** High

### Предусловие

Пользователь зарегистрирован в Trello.  
В Postman создано и выбрано окружение `Trello API Environment` с валидными значениями:

- `base_url`
- `api_key`
- `api_token`

### Шаги

1. Отправить GET-запрос:

   `{{base_url}}/members/me/boards`

2. Передать query-параметры:

   - `key={{api_key}}`
   - `token={{api_token}}`
   - `fields=name,url`

### Ожидаемый результат

1. Сервер возвращает статус `200 OK`.
2. Ответ приходит в формате JSON.
3. Тело ответа является массивом.
4. Каждый объект доски содержит поля:
   - `id`
   - `name`
   - `url`
5. Если у пользователя нет досок, возвращается пустой массив `[]`.

### Постусловие

Данные пользователя и досок не изменены.
